Custom Architecture: Cloudgarden Interiors (HTML/CSS)

View Website
GitHub Repo

To push my technical boundaries beyond drag-and-drop builders, I architected Cloudgarden Interiors—a biophilic design concept—entirely from scratch using hand-coded HTML and CSS.

Grid Mastery: While Flexbox could have simplified certain UI elements, I executed the entire site layout using CSS Grid to meet strict technical requirements and demonstrate mastery over complex, two-dimensional alignments.

Performance & Standards: The site is built with a mobile-first philosophy, ensuring fluid responsiveness across all devices. I maintained a clean, organized codebase utilizing camelCase for all JavaScript variables and CSS classes.

GitHub Integration: This project is hosted via GitHub Pages, demonstrating my ability to manage version control and move a project from a local development environment to a live production server.
